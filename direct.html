How to create direct links with this tool?

  1. First you must obtain the share URL.
    Google Drive :
    • Right click the file you want to create download link, then select share
    • Now, click on 'Copy link' or select and copy the link shown on the popup window
    • Go to the page mentioned above, and paste the link you have copied
    • Direct download link would be created without any problem

    Dropbox :
    One Drive :
  2. Here is the guide to obtain Share link.
  3. Paste that link in the 'Enter Share link' textbox.
  4. Then, press Enter or click Generate.
  5. Press Ctrl + C to copy the direct link to your clipboard.

Notes :

  • You can save this page by pressing Ctrl + S and use it whenever you need.
  • This tool will work only for Google Drive, Dropbox and Onedrive links.
