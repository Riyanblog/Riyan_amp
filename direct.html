Bagaimana cara membuat direct links dengan tool ini?

  1. Pertama-tama, anda harus tahu bagaimana untuk mendapatkan URL share dari file anda dari salah satu web-aplikasi. Hanya gunakan link dari aplikasi di bawah ini.
    Google Drive : Karena Google telah memperbarui antarmuka pengguna Google drive baru-baru ini, saya akan memberikan petunjuk untuk mendapatkan share link di tampilan antarmuka barunya. Untuk mendapatkan drive baru klik ikon kunci inggris di sudut kanan atas dari jendela drive dan pilih 'Experience the new drive'. Perlu diingat, anda tidak dapat membuat direct download Url untuk folder yang anda miliki di Google Drive dan Onedrive. Sebaliknya, anda dapat kompres folder ke dalam format .rar atau .zip dan anda dapat membuat Url yang langsung untuk file-file tersebut.
    • Klik kanan file yang ingin anda buat link download, lalu pilih share.
    • Selanjutnya, klik pada 'Copy link' atau pilih dan salin link yang ditampilkan pada jendela popup.
    • Share link sudah didapat dan bisa digunakan untuk membuat download link di tool ini.

    One Drive :
    • Klik kanan file yang anda inginkan untuk membuat URL download, kemudian pilih share.
    • Klik 'Get a link'.
    • Di tampilan 'Choose an option' pilih 'Public'.
    • Klik 'Create link'.
    • Selanjutnya, anda akan diberikan sebuah link. Gunakan itu untuk membuat direct download link di tool ini.

    Dropbox : Tidak seperti Google Drive dan Onedrive anda dapat membuat langsung Url untuk folder di dropbox. Anda dapat membuat direct Url untuk file Dropbox anda dengan mengubah angka '0' ke '1' di bagian URLnya.
    • Log in ke Dropbox.
    • Arahkan kursor pada file yang anda inginkan untuk berbagi secara langsung dan klik 'Share' dan kemudian klik 'Copy link...'
    • Gunakan link yang sudah tersalin untuk membuat link download di tool ini.
  2. Paste kan link yang didapat dari salah satu aplikasi web di atas ke dalam box 'Enter Share link'.
  3. Kemudian, tekan Enter atau klik Generate.
  4. Tekan Ctrl + C untuk menyalin direct link download ke dalam clipboard.

Catatan :

  • Anda dapat menyimpan halaman ini dengan menekan Ctrl + S dan menggunakannya kapanpun.
  • Tool ini hanya bisa digunakan untuk link dari Google Drive, Dropbox dan Onedrive.
